
# 🚀 Lumina AI – AI Study Assistant

![GitHub repo size](https://img.shields.io/github/repo-size/Pushkar012345/lumina-backend)
![GitHub stars](https://img.shields.io/github/stars/Pushkar012345/lumina-backend?style=social)
![GitHub issues](https://img.shields.io/github/issues/Pushkar012345/lumina-backend)

**Lumina AI** is a **full-stack AI-powered study assistant** that instantly generates **summaries, flashcards, and quizzes** for any topic.

**Live Demo:** [Frontend](https://lumina-frontend-ten.vercel.app/) 
**Frontend Repo link** https://github.com/Pushkar012345/lumina-frontend
---

## ✨ Features

* 🧠 Generate **AI study summaries**
* 🃏 Auto-create **flashcards**
* ❓ Generate **MCQ quizzes**
* 📜 Save and view **history**
* 🔒 Secure & fast backend (Helmet, CORS, rate limiting)

---

## 🛠️ Tech Stack

**Frontend:** React.js, Tailwind CSS
**Backend:** Node.js, Express.js, MongoDB (Mongoose)
**AI:** Groq API (LLaMA 3.3)
**Deployment:** Vercel (frontend), Render (backend)

---

## ⚡ Installation

```bash
git clone https://github.com/Pushkar012345/lumina-backend.git
cd lumina-backend
npm install
npm start
```

**Environment Variables (.env)**

```env
OPENAI_API_KEY=your_api_key
MONGO_URI=your_mongodb_connection
PORT=5000
```

---

## 📡 API Endpoints

| Endpoint               | Method | Description            |
| ---------------------- | ------ | ---------------------- |
| `/api/generate`        | POST   | Generate study content |
| `/api/history?userId=` | GET    | Get user history       |
| `/api/history/:id`     | DELETE | Delete a guide         |
